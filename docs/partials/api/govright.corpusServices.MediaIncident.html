  • search(parameters, successCb, errorCb)

    Get a media incident list by certain criteria.

    Parameters
    ParamTypeDetails
    parameters
    (optional)
    		Object

    Request parameters.

    • country – {string=} - Country code, e.g. MA, UA, etc.

    • limit – {string=} - Number of items to return.

    • skip – {string=} - Number of items to skip from the beginning of set.

    • order – {string=} - Order of the incidents (loopback order filter).

    • include – {string=} - Comma separated array of relations to include like reports,tags,nodes.

    • text – {string=} - Text string to search over incident locales.

    • date – {string=} - Date range start|end, like 2016-01-01|2016-01-02, or one of the next predefined values:
      today - all incidents for the current day,
      last-7d - for the last 7 days,
      last-30d - for the last 30 days,
      last-90d - for the last 90 days,
      last-y - for the last 365 days,
      empty value to get all incidents.

    • location – {string=} - Region id.

    • tag – {string=} - Tag id.

    • node – {string=} - Node id.

    • law – {string=} - Law id.

    (optional)
    		function(Array.<Object>,Object

    Success callback with two arguments: value, responseHeaders.

    (optional)

    Error callback with one argument: httpResponse.

    Array.<Object>
